# Skills Repository

This repository contains custom skills for AI assistants.

## Available Skills

### [ai-pm-prd](./ai-pm-prd)

A guided skill for AI Product Managers to create clear, actionable one-page PRDs for vibe coding projects.

**Use when:**
- User says "I want to build XX"
- "Help me write a PRD"
- "I need to clarify my product idea"
- "What should I tell Claude to build"
- Mention of vibe coding / AI-assisted prototyping

**What it does:**
- Guides through user persona definition
- Clarifies core features (MVP)
- Defines non-functional requirements
- Suggests tech stack
- Generates a structured one-page PRD
