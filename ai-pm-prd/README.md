# AI PM PRD Skill

一个专为 AI 产品经理设计的 Skill，帮助将模糊的产品想法快速梳理成可执行的一页纸 PRD。

## 🎯 适用场景

- 你有一个模糊的产品想法，需要把它说清楚给 AI 执行
- 你想用 Claude Code / Cursor 快速做原型，但不知道从哪开始
- 你需要一份清晰的 PRD 来指导 AI 编码助手

## ⚡ 快速开始

在 Claude Code 中，直接描述你的想法，Skill 会自动触发：

```
我想做一个工具，帮我快速整理会议笔记
```

或者直接请求：

```
帮我写个PRD，我要做一个XX
```

## 📝 Skill 做什么

- ✅ **引导式对话**：一步步引导你思考用户、功能、技术选型
- ✅ **生成一页纸 PRD**：输出结构化的 Markdown 文档
- ✅ **聚焦执行**：PRD 是给 AI 看的，不是给老板看的汇报材料

## 🚫 Skill 不做什么

- ❌ 不做"该不该做"的战略决策
- ❌ 不做市场分析和竞品调研
- ❌ 不写完整的商业 PRD
- ❌ 不写代码

## 📋 PRD 包含的内容

| 板块 | 说明 |
|------|------|
| 📋 Overview | 产品目标、为什么要现在做 |
| 👥 Target User | 目标用户画像、痛点场景 |
| 🚀 Core Features | MVP 核心功能、优先级 |
| ⚙️ Non-Functional | 性能、隐私、平台、规模 |
| 🛠️ Tech Stack | 技术栈建议及理由 |

## 🎨 交互方式

采用**引导式对话**：

1. 你说出你的想法
2. Skill 一步步提问，帮你梳理
3. 你回答问题，Skill 理解并总结
4. 完成后生成完整的 PRD

## 📦 安装

将此文件夹复制到你的 Claude Code skills 目录：

```bash
# macOS / Linux
cp -r ai-pm-prd ~/.claude/skills/

# 或直接复制整个文件夹到你的 skills 目录
```
