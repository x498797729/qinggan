# 🎯 CalmMate 演示指南

## 🚀 快速开始

### 1. 访问应用
打开浏览器访问: [http://localhost:3000](http://localhost:3000)

### 2. 体验核心功能

#### 🏠 首页概览
- 查看本周情绪统计
- 快速访问核心功能
- 查看今日建议

#### 📝 情绪签到流程
1. **点击"开始签到"按钮**
2. **选择当前情绪**（7种情绪可选）
   - 焦虑 😰
   - 压力大 😤
   - 难过 😢
   - 生气 😠
   - 平静 😐
   - 开心 😊
   - 兴奋 🤗
3. **评估情绪强度**（1-10级滑块）
4. **描述具体情况**（可选文本输入）
5. **获得AI回应和建议**

#### 📊 情绪轨迹
- 查看7天情绪变化曲线
- 了解情绪趋势和统计
- 获得个性化洞察建议

## 🔧 技术亮点

### AI 功能
- **智能共情**: 基于Kimi API的情感分析
- **个性化建议**: 根据情绪状态推荐行动
- **对话记忆**: 完整的对话流程管理

### 数据管理
- **本地存储**: 情绪数据自动保存
- **实时分析**: 情绪趋势和统计
- **可视化图表**: 使用Recharts的曲线图

### 性能优化
- **快速启动**: Next.js + Bun 极速开发体验
- **状态管理**: Redux Toolkit 高效数据流
- **类型安全**: TypeScript 完整类型检查
