🎯 产品概述

CalmMate 是一款专注大学生焦虑疏导的AI疗愈助手，通过"情绪疏导+轻量行动"帮助用户建立正向情绪循环。

🎯 核心定位

目标用户：20岁左右大学生，面临学业压力、社交焦虑、空虚感

核心价值：从"用户不问AI不答"进化到主动关怀的疗愈体验

差异化：专注垂直细分，避免泛陪伴陷阱

📊 市场背景与机会

行业痛点分析

🔴 情感真实性缺失

87%用户认为AI缺乏真实情感连接，像复读机一样机械化

🧠 记忆能力不足

AI记不住对话历史，每次都要重新介绍基本信息

⚡ 技术故障频发

应用崩溃、响应延迟严重影响使用体验

📉 留存率低下

多数用户1-2个月即流失，月使用不足5天

产品机会窗口

✅ 市场空白

  • 真实情感AI伴侣：深度情感理解+个性化成长
  • 长期记忆系统：持续个人化体验
  • 垂直细分定位：专注大学生焦虑疏导

🚀 MVP功能规格

🎯 MVP核心三件套

宁可只留3个功能做到极致，也不要7个功能每个都半吊子

📝 极简情绪签到对话

核心价值：3-5分钟结构化短对话，快速情绪收集+AI共情回应

验收标准：90%用户反馈"感觉被理解"

🎯 微行动推荐系统

核心价值：基于情绪状态推荐3种基础行动（呼吸/感恩/运动）

验收标准：用户完成率 > 60%

📊 情绪轨迹可视化

核心价值：7天情绪变化曲线，让用户看到自己的进步

验收标准：建立正向反馈循环

✅ MVP包含功能

  • 极简情绪签到：3分钟快速情绪记录+AI回应
  • 微行动推荐：3种基础行动类型+简单指导
  • 情绪轨迹图：7天情绪变化可视化

❌ MVP排除功能

  • 深度对话系统（V1.1+）
  • 复杂记忆功能（V1.1+）
  • 多维情绪标签（V1.1+）
  • 个性化推荐算法（V1.1+）
  • 月度报告（V2.0+）
  • 匿名社区（V2.0+）
  • 危机干预（V2.0+）

🏗️ 技术架构

前端技术栈

框架：React Native + TypeScript
UI库：React Native Paper
状态管理：Redux Toolkit
本地存储：AsyncStorage + SQLite

后端技术栈

语言：Node.js + TypeScript
框架：Express.js
数据库：SQLite（初期）→ PostgreSQL（后期）
AI服务：OpenAI API + 自定义Prompt工程
部署：Vercel + Railway

核心数据模型

actions user_id, action_type, mood_before, mood_after 行动记录

🎨 UI/UX设计规范

设计原则

✨ 设计理念

  • Calm App的留白感 + flomo的轻量记录感
  • 界面要有呼吸感，不拥挤，一眼看到核心内容
  • 绝对不要暗色系、赛博风、二次元风格

色彩系统

🎨 主色调

温暖的咖啡棕色 (#8B7355)
营造安全、温暖的氛围

💙 辅助色

雾霾蓝 (#7BA0B8)
增加清新感和专业度

🤍 背景色

温暖的米白色 (#FEFCF8)
提供舒适的阅读体验

核心界面规范

界面 设计重点 用户体验目标
情绪签到 极简输入，引导式问题 3分钟内完成情绪记录
对话界面 舒适气泡，充足留白 让用户感到被倾听和理解
情绪轨迹 美观曲线，直观数据 看到自己的进步和变化

第10周+：正式发布

小范围校园推广，监控核心指标，持续迭代
